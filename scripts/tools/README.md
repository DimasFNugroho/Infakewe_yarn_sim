# Tools

Helper scripts for local development and reproducibility.

- `setup_env.*` creates the conda environment from base/lock/explicit specs.
- `run_local_tests.*` runs pytest inside the conda env.
- `smoke_test.py` is a minimal proof-of-life demo for PyChrono (+ Irrlicht).
- `freeze_env.*` exports lock/explicit specs from an existing env.
- `install_conda.sh` installs Miniconda quickly on Linux.
